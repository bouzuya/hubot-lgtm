# hubot-lgtm

A Hubot script for fetching an image from www.lgtm.in.

## Usage

    $ npm install git://github.com/bouzuya/hubot-lgtm.git

or

    $ # TAG is the package version you need.
    $ npm install git://github.com/bouzuya/hubot-lgtm.git\#TAG
