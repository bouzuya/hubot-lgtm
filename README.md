# hubot-lgtm

A Hubot script for fetching an image from www.lgtm.in.

## Usage

    $ npm install https://github.com/bouzuya/hubot-lgtm/archive/master.tar.gz

or

    $ npm install https://github.com/bouzuya/hubot-lgtm/archive/{VERSION}.tar.gz
